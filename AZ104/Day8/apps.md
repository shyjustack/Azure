### What is Azure Container Apps?

Azure Container Apps (ACA) is a fully managed, serverless platform for running containerized applications without managing VMs or Kubernetes clusters.

It is built on top of Azure Kubernetes Service (AKS) internally, but you never need to manage Kubernetes.

### What Azure Container Apps offers

- Run containerized microservices
- Autoscaling (including scale to zero)
- Built-in HTTPS ingress
- Dapr support (service-to-service communication, pub/sub, state)
- Background jobs & event-driven apps
- Revisions for Blue/Green deployments
- Sidecar containers
- VNET integration
